# InvestIQ — AI Investment Research Agent

An AI-powered investment research platform that helps users analyze companies and make informed investment decisions. Enter a company name, and InvestIQ gathers insights from multiple perspectives—including financial performance, recent news, market sentiment, competitive positioning, and potential risks—to generate a clear **Invest**, **Watchlist**, or **Pass** recommendation.

Rather than displaying raw AI output, the platform organizes the analysis into a clean, interactive dashboard with investment scores, financial charts, competitor comparisons, risk breakdowns, and an easy-to-understand investment thesis. Every recommendation is backed by supporting insights, making the decision process transparent and easy to follow.

## ✨ Features

* 🤖 Multi-agent AI workflow for comprehensive company research
* 📊 Investment score based on multiple evaluation factors
* 📈 Interactive financial charts and business metrics
* 📰 News and market sentiment analysis
* 🐂🐻 Bull vs. Bear investment debate
* 🛡️ Risk assessment with categorized insights
* 📄 Exportable investment report
* 👶 "Explain Like I'm Five" mode for simplified explanations
* ⚡ Live analysis progress with streaming updates
* 🌙 Modern responsive dashboard with dark mode

## 🏗️ Architecture

The application is built using **Next.js**, **LangGraph.js**, and **OpenAI GPT-4o**.

The analysis is divided into specialized AI agents, each responsible for a specific task:

* Research Agent – Collects company information and business overview
* Financial Agent – Evaluates revenue, profitability, valuation, and growth
* News Agent – Summarizes recent developments
* Sentiment Agent – Analyzes market and investor sentiment
* Competitive Agent – Compares the company with industry peers
* Risk Agent – Identifies key business and market risks
* Decision Agent – Combines all findings into a final recommendation

Using LangGraph allows each agent to contribute to a shared state, making the workflow modular, easy to extend, and simple to maintain.

## 🚀 Running the Project

```bash
git clone <repository-url>
cd investiq
npm install
cp .env.local.example .env.local
npm run dev
```

Add your OpenAI API key to `.env.local` before starting the application.
